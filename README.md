# E-commerce Analytics Dashboard

## Project Overview

This project presents an interactive E-commerce Analytics Dashboard developed using Power BI and Tableau to analyze sales performance, customer behavior, profitability, payment trends, shipping metrics, and product performance.

## Tools Used

- Power BI
- Tableau
- Microsoft Excel
- Power Query
- DAX

## Skills Demonstrated

- Data Visualization
- Dashboard Design
- Customer Analytics
- Sales Analytics
- Data Modeling
- DAX
- Power Query

## Dashboard Features

- Sales KPIs
- Revenue Analysis
- Profit Analysis
- Product Performance
- Customer Insights
- Shipping Performance
- Review Score Analysis
- Payment Analysis
- Geographic Analysis

## Key Insights

- Monitor sales performance.
- Identify profitable products.
- Analyze customer purchasing behavior.
- Evaluate shipping efficiency.
- Track payment trends.
